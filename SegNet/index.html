SegNet with Keras library


The task is to use SegNet neural network in order to do image segmentation. There are 13 classes: Sky, Building, Pole, Road_marking, Road, Pavement, Tree, SignSymbol, Fence, Car, Pedestrian, Bicyclist, Unlabelled. The code can be find here.

Methods

First of all, the images are download from their files (one for train and one for test). It is the same step for the labels. The colors are then chosen for each labels. Then the model has been trained.
