AiSight Challenge


Two datasets of vibration have been recorded from two different machines using our AiSight Sensor-node. Each dataset has 5 columns: Unix time, Amount of samples, Time period (milliseconds), Sampling Rate, Sensor data (millivolt reads). The goals are to identify different machine states in the data set and cluster them and to come up with a classification algorithm that classifies the states into: State 1, 2, 3 etc...

Methods

First of all, there is no column names in the datasets and the fifth column cantains lists. The first step was to put columns names without losing the first row of the dataset. Next, it has been necessary to replace the lists by numbers. In order to get that, I did the average value of each list. Then, 10 machine learning techniques have been compared for the prediction of the state machine. Two studied have been done, one for each dataset.
