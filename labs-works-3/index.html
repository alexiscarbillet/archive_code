Subject


This assignment aims to design an effective multi-label multi-source classifier and then infer demographics of users based on a given dataset collected from multiple social networks. Each dataset contains 10 columns: row ID, educationInfoForAgeGroupEstimation, workInfoForAgeGroupEstimation, gender, realAge, ageGroup, relationship, educationLevel, occupation, income.

Methods

First of all, 3 different datasets have been studied. Each of those dataset was not completed. There were some empty parts. They have been filled with "-1". Then the information have been conveted in integer. Then, 14 machine learning techniques have been compared for the prediction of label desired. Three studied have been conducted in order to know which model was the more efficient for which dataset. Finally, a late fusion has been done.

Results

In this part, the results are presented:

The gender has been predicted in this part. The best model for London is kNN 15 neighbors with a F1 score of 0.603. For New York it is the random forest with 0.667 as F1 score. The best prediction for Singapore is obtained with Gradient Boosting and a F1 score of 0.565
